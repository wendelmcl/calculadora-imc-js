# Calculadora de IMC

Projeto desenvolvido com HTML, CSS e JavaScript para cálculo do Índice de Massa Corporal (IMC), apresentando classificação automática e interface moderna responsiva.

## Objetivo

Praticar conceitos fundamentais de desenvolvimento front-end, incluindo estruturação de páginas, estilização moderna, manipulação do DOM e lógica com JavaScript.

## Funcionalidades

- Cálculo automático do IMC
- Classificação por categorias
- Atualização dinâmica dos resultados
- Interface moderna e responsiva
- Limpeza automática dos campos
- Mensagens informativas conforme o resultado

## Categorias do IMC

- Abaixo do peso
- Peso normal
- Sobrepeso
- Obesidade grau I
- Obesidade grau II
- Obesidade grau III

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Conceitos praticados

- Manipulação do DOM
- Eventos com JavaScript
- Condicionais
- Funções
- Captura de dados de formulários
- Atualização dinâmica de elementos HTML
- Layout com Flexbox
- Gradientes e sombras modernas
